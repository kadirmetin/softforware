# Softforware
Bu proje, Softforware için geliştirilmiş bir React tabanlı web blog projesidir. Amacı, Softforware sitemizin ziyaretçilerine yazılım hakkında bilgi sağlamaktır. Projede, kullanıcılar yazıları okuyabilir ve yorum yapabilirler.

Proje, Node.js ve Express ile birlikte çalışır ve MongoDB veritabanını kullanır. Projenin çalıştırılması için, Node.js ve MongoDB yüklü olmalıdır.
