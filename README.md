# Softforware
Bu proje, Softforware için geliştirilmiş bir NextJS tabanlı web blog projesidir. Amacı, Softforware sitemizin ziyaretçilerine yazılım hakkında bilgi sağlamaktır. Projede, kullanıcılar yazıları okuyabilir ve yorum yapabilirler.

Proje, NextJS ile birlikte çalışır ve Supabase veritabanını kullanır. Detaylı kurulum rehberi daha sonra eklenecektir.
